Privacy & Security Help Privacy & Security Data we collect This app stores inventory records such as item name, category, quantity, price, supplier name, stock status, featured flag, and optional notes. Personal user identity data is not required for basic inventory operations. How data is used Inventory data is used only to support list, search, create, update, and delete functions. Users should enter only business-related information and avoid placing sensitive personal details in the special note field. Security controls The app communicates with the server through HTTPS to protect data in transit. Input validation is applied for required fields and numeric values to reduce invalid or unsafe data submissions. User responsibilities Keep your mobile device updated, use device lock protection, and avoid using public unsecured networks when managing inventory records. Report suspicious changes in records to your team immediately.
